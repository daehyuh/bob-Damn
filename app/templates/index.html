🔓 취약한 웹 애플리케이션

⚠️ 경고: 이 애플리케이션은 AWS 보안 테스트 목적으로 의도적인 보안 취약점을 포함하고 있습니다. 제품 환경에서는 절대 배포하지 마세요!

사용자 관리 API

POST/api/v1/users/login [SQL 인젝션 취약점]
POST/api/v1/users/register [SQL 인젝션 취약점]
GET/api/v1/users/profile/{user_id} [SQL 인젝션 취약점]
GET/api/v1/users/search?q=query [SQL 인젝션 취약점]

파일 관리 API

POST/api/v1/files/upload [S3 CloudTrail 이벤트 생성]
GET/api/v1/files/download?filename= [디렉토리 탐색 위험]
POST/api/v1/files/bulk-upload?count= [S3 대량 작업]

관리자 API (권한 상승)

GET/api/v1/admin/users?user_id= [IDOR - 다른 사용자 정보 접근]
POST/api/v1/admin/sudo-command [약한 패스워드로 sudo 권한 획득]
POST/api/v1/admin/change-user-role [약한 키로 사용자 역할 변경]
POST/api/v1/admin/create-instance [EC2 인스턴스 생성 - CloudTrail 이벤트]

RDS 공격 API (데이터베이스 보안 테스트)

POST/api/v1/rds/brute-force-attack [RDS 무차별 대입 - CloudWatch 알람]
POST/api/v1/rds/sql-injection-mass-query [대량 SQL 인젝션 - RDS 로그]
POST/api/v1/rds/connection-exhaustion [연결 풀 고갈 - CloudWatch]
GET/api/v1/rds/rds-performance-impact [리소스 집약적 쿼리 - Performance Insights]
POST/api/v1/rds/create-rds-instance [취약한 RDS 생성 - CloudTrail 이벤트]

익스플로잇 API (테스트용)

GET/api/v1/exploit/ssrf?url= [서버 측 요청 위조]
GET/api/v1/exploit/command-injection?cmd= [명령어 인젝션]
POST/api/v1/exploit/port-scan [네트워크 스캔 - VPC Flow Logs]
POST/api/v1/exploit/crypto-mining-sim [CPU 사용률 스파이크 - CloudWatch 메트릭]
GET/api/v1/exploit/metadata-access [AWS 메타데이터 서비스 접근]
